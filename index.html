WebRTC P2P Game

Instructions:

  • Player 1 (Host): Click "Create Game" to generate an invite code. Copy and send it to Player 2.
  • Player 2: Paste Player 1's code in "Paste Invite Code" and click "Join Game". Copy the response code and send it back to Player 1.
  • Player 1: Paste Player 2's response in "Paste Response Code" and click "Connect".
  • Controls:
    • PC: A (left), D (right), W (jump)
    • Mobile: Use touch buttons below
